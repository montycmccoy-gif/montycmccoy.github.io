# ClawBot Fleet

**The digital co-pilot that autonomizes vehicle-dependent daily tasks beyond FSD/Robotaxi.**

Built exclusively with Grok 4.2 isolated core on a MacMini.  
Real Tesla Fleet API + Robotaxi/Cybercab integration.  
Phone-controlled via Telegram.  
Reclaims hours by turning natural-language requests into parallel fleet errands.

### What it does today
- Real curbside coffee runs (Philz example) with precondition, summon, trunk open
- Multi-Cybercab optimization (12+ simultaneous runs)
- Live financial reporting from Tesla Fleet API
- Role-based dashboard (Ops Manager, Accountant, Driver)
- Full phone control via Telegram bot

### Live demo
Open the dashboard: https://montycmccoy-gif.github.io/clawbot_fleet_dashboard.html  
(Uses your own Tesla Bearer token — safe, local-only)

### How to run locally
1. Clone this repo
2. Place your Tesla PEM and callback.html (already here)
3. Run the PKCE auth script to get Bearer token
4. Open `clawbot_fleet_dashboard.html`
